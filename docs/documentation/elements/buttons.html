A Simple Button

Initializing a button is really easy. Just add class name of 'button' to the span tag


Button 
<span class="button">Button</span>

There are lot of other ways to create a Button
  • You can use the
    button
    class with anchor tags
    <a></a>
    or with div tags
    <div></div>
    or with span tags
    <span></span>
    .
  • The button tag
    <button> </button>
    can be used directly
  • For forms, you can use the input tag
    < input class="button" >
    .
